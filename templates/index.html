Command Repository Built for use with INNBot, by NYXT.

How does this work?

Command Repository makes use of INNBot's repository command - this allows INNBot to fetch, every 5 minutes or so, a list of simple single-response commands from a website that it can use. Using this application, you'll be able to easily add and remove repository commands.

How do I use it?

It's easy. If you have INNBot running, just add our JSON link to your configuration. If you want to add a command, just fill in Add a Command form below and press Submit, you'll be up and running in no time at all*.

* "No time at all" is approximately 5 minutes.

How was it built?

INNBot was built usng NodeJS and Discordie - a really nice wrapper for the Discord Official API - it's fully open-source and you can check that out here. This Command Repository is an example application that demonstrates how you can make use of INNBot's repository functionality - it's built using Flask, Flask-SQLAlchemy and Skeleton for styling.
